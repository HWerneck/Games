•-- Português --•
Olá a todos!
O Bankrupt foi compilado e testado num sistema Windows 10, usando o terminal PowerShell e o MinGW como o compilador.
Para compilação, é necessário estar no diretório principal do jogo, onde se encontram todos os arquivos do jogo.
O comando utilizado para compilar o jogo é:

	g++ *.cpp -o Bankrupt.exe

Depois disso, o arquivo "Bankrupt.exe" é criado no mesmo diretório e é executado ao ser acionado.
O jogo Bankrupt pode ser acionado diretamente do PowerShell com o comando:

	.\Bankrupt.exe

Ao término da execução, um arquivo .csv é gerado com dados referentes às partidas jogadas.



•-- English --•
Hey there!
Bankrupt was compiled and tested in a Windows 10 system, using terminal PowerShell and MinGW as the compiler.
To compile, it is necessary to be in the main game directory, where all the game files are.
The command used to compile the game is:

	g++ *.cpp -o Bankrupt.exe

After that, the file "Bankrupt.exe" is created in the same directory and is run when activated.
The Bankrupt game can be activated directly from PowerShell with the command:

	.\Bankrupt.exe

At the end of execution, a .csv file is generated will the data referring to the matches played.
